# 轮播图文件夹

## 说明
此文件夹用于存放轮播图图片。

## 图片要求
- 格式：JPG、PNG
- 尺寸：建议 1200x400 像素
- 数量：5-8张为宜

## 使用方法
1. 将图片放入此文件夹
2. 修改 `assets/js/views/chatView.js` 中的 `carouselData` 数组
3. 更新 `index.html` 中的轮播图初始化代码

## 示例图片命名
- carousel1.jpg
- carousel2.jpg
- carousel3.jpg
- carousel4.jpg
- carousel5.jpg

## 生成纯色占位图
由于技术限制，当前使用纯色占位图。您可以：
1. 打开浏览器访问以下页面生成图片：
   - `generate_carousel1.html` - 明亮蓝
   - `generate_carousel2.html` - 柔和粉
   - `generate_carousel3.html` - 现代紫
   - `generate_carousel4.html` - 医疗蓝
   - `generate_carousel5.html` - 医美粉

2. 或者使用其他图片编辑工具创建纯色图片

3. 将生成的图片重命名为对应的文件名并放入此文件夹
