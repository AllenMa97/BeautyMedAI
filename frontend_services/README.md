# YISIA 前端服务

这是一个简单的静态文件服务器，用于部署和分享YISIA前端应用。

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动服务器
```bash
npm start
```

服务器将在 `http://localhost:3000` 上运行。

## 局域网访问

如果您想让同一局域网内的其他人访问此服务，请使用以下地址：
```
http://<您的IP地址>:3000
```

其中 `<您的IP地址>` 是运行此服务器的计算机的局域网IP地址。

## 访问应用

启动服务器后，您可以通过以下方式访问应用：

- 本地访问：http://localhost:3000
- 局域网访问：http://<your-ip-address>:3000

## 停止服务器

按 `Ctrl+C` 停止服务器。

## 功能特性

- 静态文件服务：提供所有前端资源文件
- SPA支持：支持单页应用程序的路由
- 跨域访问：允许局域网内其他设备访问
- 热更新：实时提供最新文件

## 注意事项

- 确保防火墙允许3000端口的访问
- 如果需要其他设备访问，请确保网络设置允许
- 服务器默认监听所有网络接口 (0.0.0.0)
